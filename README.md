# WP-CLI User Manager 🧑‍💻

A simple command-line tool to manage WordPress users using WP-CLI.

## 🔧 Features

- List all users
- Create a user
- Delete a user
- Change user role
- Reset password

## 📦 Requirements

- WordPress installed locally
- [WP-CLI](https://wp-cli.org/) installed and configured

## 🚀 Usage

```bash
./user-manager.sh
